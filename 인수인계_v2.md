## 2. 방향 전환 — 사용자 확정 결정 (전부 승인됨)

사용자가 픽셀건(복셀 슈터) '깃발 꽂기' 모드를 참조한 **팀 대전 게임**으로 전환을 지시했고,
선택지 질문으로 아래 4개를 확정받았다(모두 사용자가 명시 선택 — 재확인 불필요):

1. **신규 앱** — 기존 저장소 내 `ChemVerse/arena/` 폴더. 본편 파일은 일절 수정 금지(보존)
2. **3D 복셀 TPS** — 블록형 캐릭터 + 어깨너머 3인칭 (1인칭 FPS·2D 탑다운 기각)
3. **네트워크 이중화** — 수업 본번 = 교내망 wsrelay(교사 PC), 시연·과제 = Supabase. 어댑터 구조 유지
4. **이번 세션 범위 = 지시서 + Phase A 기반 구현**(로그인·미션선택·팀매칭·복셀TPS·맵·8Hz 동기화).
   전투·부품·조립·퀴즈는 Phase B(다음 이후)

모델 정책(사용자 지시 + CLAUDE.md): **Fable = 설계·지시서·계약·검수 / Sonnet = 구현 하위 에이전트**.

게임 규칙 원본(사용자 구상 — 전부 `arena/docs/GAME_DESIGN.md`에 반영 완료, 원문 재수집 불필요):
부품 조립로 미션 완수 시 승리(깃발꽂기 대응) / 제한시간·점수 비교 / 실험기구 무기(뷰렛·스포이트 등),
킬 시 상대 3초 리스폰+부품 강탈 가능하나 **본인 점수 하락(연구윤리 위반)** / 게임 후 실험 세팅
기반 실제 현상 관찰(과학 법칙 정합). 교과서 '물질과 에너지' 20쪽 PDF의 준비물 6종

수식 M = wRT/PV도 GAME_DESIGN §3에 반영됨 — **PDF 재업로드 불필요**.
