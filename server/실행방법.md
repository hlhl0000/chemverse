# ChemVerse 로컬 서버 실행 방법 (교사용 PC)

1. Node.js LTS 버전을 설치합니다 (https://nodejs.org).
2. cmd 창을 열고 `cd 경로\ChemVerse` 로 프로젝트 폴더까지 이동합니다.
3. 최초 1회만 `npm install ws` 를 실행해 의존성을 설치합니다.
4. `node server/relay.js` 를 실행하면 8080 포트로 서버가 켜집니다.
5. 학생들은 같은 Wi-Fi에서 브라우저로 `http://이 PC의 IP:8080` 에 접속합니다 (교사 PC 자신은 `http://localhost:8080`).
