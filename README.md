# ChemVerse — 웹 기반 3D 가상 과학실

용인고등학교 화학·통합과학 수업용. **앱 설치 없이 URL만으로** 태블릿에서 접속하는 3D 메타버스 실험실.

## 학생이 하는 일
1. URL 접속 → 이름·방 코드 입력 → 3D 실험실 입장 (같은 방 코드끼리 서로의 아바타가 보임)
2. 가상 조이스틱으로 이동, 화면 드래그로 시점 회전
3. 실험대 콘솔 탭 → 실험 선택 → **선반에서 기구를 직접 골라 실험대에 배치** (자유 세팅)
4. 장치 완성 후 히터·밸브·피스톤 조작 → P/V/n/T 수치와 실시간 그래프 관찰
5. 포털로 자연 관찰장 이동 → 고도별 기압과 풍선 부피 변화 관찰

## 실험 목록
| 실험 | 상태 |
|---|---|
| 이상기체 방정식 (PV=nRT, 등온·등압·등적) | ✅ 완성 |
| 라울 법칙 (증기압 내림) | 🔧 설계 완료, 구현 대기 |
| 반응 속도 (시계 반응) | 🔧 설계 완료, 구현 대기 |

## 실행·배포
- 로컬 확인: 폴더에서 `python -m http.server 8000` → http://localhost:8000 (파일 직접 열기는 ES 모듈 때문에 불가)
- 배포·멀티플레이 설정: **docs/DEPLOY.md** 참고 (GitHub Pages + Supabase 무료 / 교내망 릴레이)
