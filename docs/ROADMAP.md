# ChemVerse 로드맵

## 3일 범위 판정 (Fable 검토 결론)

MEL VR 수준의 "풀 멀티플레이 메타버스 + 다수 실험"을 3일에 완성하는 것은 **불가**.
그러나 아래 Phase 1은 3일 내 실현 가능하며, 이후 확장이 계약(인터페이스)만 지키면 되도록 토대를 설계했다.
